# Prot Skills v0.0.3

This release formalizes the desktop packaging flow and keeps the project ready
for repeatable GitHub draft releases across macOS, Linux, and Windows.

## Highlights

- Standardized release automation for tagged desktop builds.
- Keeps application version metadata aligned across the frontend and Tauri app.

## Added

- Version bump and tag helper scripts for the release workflow.
- Multi-platform GitHub draft releases powered by Tauri Action.

## Changed

- Release publishing now runs from versioned `v*` git tags.

## Fixed

- Build metadata synchronization checks now protect release packaging from
  mismatched app versions.

## Known Issues

- macOS builds are not signed yet. If Gatekeeper blocks the app, remove the
  quarantine attribute:

  ```sh
  xattr -d com.apple.quarantine "/Applications/Prot Skills.app"
  ```

## Download Notes

- Download the asset that matches your platform from the release assets below.
- If you are updating from an older local build, back up important managed
  Skills before testing migration-related workflows.
