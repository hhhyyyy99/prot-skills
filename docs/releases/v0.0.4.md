# Prot Skills v0.0.4

This release upgrades the desktop publishing workflow so each version ships with
clear release notes, more recognizable download filenames, and a more reliable
tag-driven draft release process.

## Highlights

- GitHub draft releases now use versioned release notes stored in the
  repository.
- Release assets now upload with clearer product-style filenames.

## Added

- A release notes template for future tagged desktop releases.
- Repository documentation for the full `main -> version bump -> tag` release
  flow.

## Changed

- Tagged releases now fail early if the matching `docs/releases/vX.Y.Z.md` file
  is missing.
- GitHub release assets now use names such as
  `Prot-Skills-v0.0.4-macos-aarch64.dmg`.

## Fixed

- The Chinese README now reflects the current Tauri 2.x stack instead of the
  old 1.x wording.

## Known Issues

- macOS builds are not signed yet. If Gatekeeper blocks the app, remove the
  quarantine attribute:

  ```sh
  xattr -d com.apple.quarantine "/Applications/Prot Skills.app"
  ```

## Download Notes

- Download the asset that matches your platform from the release assets below.
- Windows and Linux packages may use different installer formats depending on
  the build target produced by Tauri.
