# Prot Skills v0.0.6

Patch release with deletion UX improvements and CI enhancements.

## Highlights

- Improved page deletion experience with better responsiveness during database migrations.
- Added AI review toggle for CI pipelines.

## Fixed

- Page deletion UX and migration responsiveness (#18).

## Changed

- Added `AI_REVIEW_ENABLED` repository variable to skip AI review in CI workflows (#17).

## Known Issues

- macOS builds are not signed yet. If Gatekeeper blocks the app, remove the
  quarantine attribute:

  ```sh
  xattr -d com.apple.quarantine "/Applications/Prot Skills.app"
  ```

## Download Notes

- Download the asset that matches your platform from the release assets below.
- Windows and Linux packages may use different installer formats depending on
  the build target produced by Tauri.
