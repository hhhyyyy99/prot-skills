# Prot Skills v0.0.8

Patch release with broader tool detection, update checks, and safer skill
migration handling.

## Highlights

- Added detection support for Pi CLI and Gemini CLI skill directories.
- Improved migration and sync reliability so lifecycle errors are reported
  clearly instead of being dropped or merged into generic failures.

## Added

- Manual update check from the settings page.
- Skill descriptions from `SKILL.md` frontmatter in the skills list and
  migration page.
- Operation reports for skill lifecycle actions.

## Changed

- Split page business logic into feature modules to keep page components
  smaller and easier to maintain.
- Added local agent rules and a self-check workflow for safer development and
  PR validation.

## Fixed

- Restores original skill folders when symlink installation fails.
- Handles managed duplicate migrations safely.
- Removes completed migration candidates from the list.
- Shows failed skill names in sync errors and reports migration sync issues
  separately.
- Removes silent error dropping and panic paths from sync link operations.
- Renders the titlebar spacer only on macOS.

## Known Issues

- macOS builds are not signed yet. If Gatekeeper blocks the app, remove the
  quarantine attribute:

  ```sh
  xattr -d com.apple.quarantine "/Applications/Prot Skills.app"
  ```

## Download Notes

- Download the asset that matches your platform from the release assets below.
- Windows and Linux packages may use different installer formats depending on
  the build target produced by Tauri.
